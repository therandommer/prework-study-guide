HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The footer element contains elements such as navigation and other information.
  • Use "ul" for unordered lists. "ol" for ordered lists. "li" to represent list items.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can refer to classes defined in the html file using .classname to apply CSS styling just to that class.
  • The * attribute can be used to refer to all visible elements.

Git

  • Ensure git commits are made frequently to provide a backup of the work completed.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use 'var' to declare a variable, use 'function' to declare a function.
  • Math.Random(Floor, Ceil) is very useful for creating random generation.
